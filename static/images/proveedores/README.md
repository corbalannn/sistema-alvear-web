# Logos de Proveedores

Esta carpeta contiene los logos de los proveedores textiles.

## Archivos esperados:

1. **tn_platex.png** - Logo de T&N Platex
   - Formato recomendado: PNG con fondo transparente
   - Tamaño sugerido: 120x60px o mantener proporción

2. **tecotex.png** - Logo de Tecotex S.A.
   - Formato recomendado: PNG con fondo transparente
   - Tamaño sugerido: 120x60px o mantener proporción

3. **emilio_alal.png** - Logo de Emilio Alal S.A.
   - Formato recomendado: PNG con fondo transparente
   - Tamaño sugerido: 120x60px o mantener proporción

## Instrucciones:

1. Descarga los logos reales de cada empresa
2. Renómbralos según los nombres de archivo especificados arriba
3. Colócalos en esta carpeta
4. Los logos aparecerán automáticamente en el formulario de ingreso

## Formatos soportados:
- PNG (recomendado para logos con transparencia)
- JPG/JPEG
- SVG
- WebP
